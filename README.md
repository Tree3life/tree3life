# Tree3Life🌱

## 简介

本项目是一个可以用于前后端分离web开发的==`架子`==，同时也是一个 整合了个人编程思维结构的\`聚合项目\`。项目创建的初衷——不以业务为参照，以建立自己的Java后端开发体系为目标。

项目内容：

- 🌱沉淀本人在开发中 涉及到的、掌握的(重构 归纳成模板/工具类)的技术，记录计划中要学习掌握的技术。
- 🌱记录/实现`个人对`Java前后端分离模式`下的各个编程模块的构思。

## 项目技术

- 前端
  - React、Redux、Router、AntD UI、
- 后端
  - Springboot
  - MySQL 
  - Redis
  - RabbitMQ
